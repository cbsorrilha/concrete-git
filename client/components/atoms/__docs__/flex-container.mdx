# Flex Container
The Flex Container give you options to control the flex properties of that element:


## Inline
This props create a `display: inline-flex` element

<FlexContainer inline={true} style={{border: 'solid black thin'}}>
  <FlexItem>
    Flex Item 1
  </FlexItem>
  <FlexItem>
    Flex Item 2
  </FlexItem>
  <FlexItem>
    Flex Item 3
  </FlexItem>
  <FlexItem>
    Flex Item 4
  </FlexItem>
  <FlexItem>
    Flex Item 5
  </FlexItem>
</FlexContainer>
